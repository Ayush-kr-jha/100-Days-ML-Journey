# Data Gathering

This section covers different methods of collecting data for Machine Learning
using Python and Pandas.

## Topics Covered

- Working with CSV files
- Working with JSON & SQL data
- Fetching data from APIs
- Web scraping

## Folder Overview

- `working_with_csv/`  
  Reading CSV files using Pandas and understanding important parameters
  like `sep`, `header`, `index_col`, `usecols`, `dtype`, `parse_dates`,
  `na_values`, and handling large files using `chunksize`.

- `working_with_json_sql/`  
  Working with JSON files and loading data from SQL databases.

- `fetching_data_from_api/`  
  Collecting data from external APIs using HTTP requests.

- `fetching_data_using_web_scraping/`  
  Extracting data from websites using web scraping techniques.

> Each folder contains notebooks with explanations and hands-on examples.
