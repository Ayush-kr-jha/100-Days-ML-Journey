# ML Fundamentals Folder

This folder contains notes and documentation on **core Machine Learning concepts**.  
The focus is on understanding the fundamentals before moving to coding and projects.



## Files

### applications_of_machine_learning.md
- Real-world applications of ML across different industries  
- Examples in Retail, Healthcare, Finance, Transportation, Manufacturing, Marketing  
- Quick reference for practical use cases  

### challenges_in_machine_learning.md
- Common challenges while working with ML  
- Topics include data collection, quality, insufficient data, overfitting/underfitting, model selection  
- Notes on computational resources, interpretability, and deployment issues  

### instance_vs_model_based_learning.md
- Comparison of **instance-based vs model-based learning**  
- Key characteristics, storage, prediction methods  
- Examples like KNN, Linear Regression, Neural Networks  
- Quick table for differences and when to use each  

### ml_development_life_cycle.md
- Step-by-step overview of building an ML system  
- From problem definition → data collection → preprocessing → feature engineering → training → evaluation → deployment → monitoring  
- Focused on the workflow, not implementation  

### tensors_basics.md
- Introduction to **scalars, vectors, matrices, and higher-dimensional tensors**  
- Rank, shape, and size of tensors  
- Examples in Python/NumPy  
- Summary table for quick reference
